Vulkan Text Library

bruvtext

`bruvtext` is a vendorable text library for Vulkan projects. It shapes text with HarfBuzz, rasterizes glyphs with FreeType, caches glyph atlases, and provides draw data for a host renderer.

Public API

DrawText

Simple draw and measure calls. No public cache knobs.

Vulkan

Bundled showcase and benchmark demo using the real GPU path.

What It Is

Scope

  • Vulkan-oriented text library with a normal C++ API.
  • Host renderer keeps ownership of Vulkan images, buffers, descriptors, and command submission.
  • Library owns shaping, glyph caching, atlas contents, and draw-data generation.
  • Bundled demo acts as the reference renderer and benchmark.
This is not a UI framework and it is not trying to be CSS. It is a text system that can be integrated into an existing Vulkan renderer.
