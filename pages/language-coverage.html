Language Coverage

What the showcase proves

The demo ships 19 sample languages. That is a validation set, not a hard support boundary. Actual support depends on the registered fonts and the script behavior covered by HarfBuzz and FreeType.

Sample Set

Bundled demo languages

English, French, Spanish, German, Italian, Portuguese, Dutch, Vietnamese, Polish, Turkish, Romanian, Czech, Swedish, Japanese, Simplified Chinese, Traditional Chinese, Korean, Arabic, and Hindi.

What Determines Support

Real boundary conditions

  • Does the font contain the glyphs the text needs?
  • Does HarfBuzz shape the script correctly?
  • Can FreeType rasterize the selected glyphs at the requested sizes?
  • Can the host tolerate the atlas memory required for the glyph set in use?

Fonts

Library support does not equal bundled font coverage

The demo includes fonts needed to exercise several script families. A host project is still expected to register fonts that match its own language requirements. If a chosen font does not contain a glyph, the result will be missing glyph output from that font.
